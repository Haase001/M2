Contenido

A lo largo de este proyecto aprendimos como usar CSS para crear el diseño de una página web,
Nuestro aprendizaje abarco diversos temas con los cuales pudimos lograr crear esta pequeña página web:

  • Reglas en CSS
  • Propiedades Básicas
  • Displays
  • Hover
  • Box Model
  • Herencia y Especificidad
  • Display: Flexbox
  • Positions
  • Display: Grid
  • Estilos Responsivos

Reglas en CSS

Las reglas en CSS consisten en un selector y un bloque de declaraciones encerrado entre llaves, donde cada declaración define una propiedad y su valor para aplicar estilos a elementos HTML.

Propiedades Básicas

Incluyen color, font-family, font-size, margin, padding, y background, que controlan la apariencia de texto, espaciado, y fondo de los elementos.

Displays

Define cómo se organizan los elementos en la página: block, inline, inline-block, o none, afectando su flujo y visibilidad.

Hover

Es un pseudo-clase que aplica estilos cuando el usuario pasa el cursor sobre un elemento, permitiendo efectos interactivos.

Box Model

Es el modelo que describe cómo se estructura un elemento: contenido, padding, borde y margen, afectando su tamaño y posición.

Herencia y Especificidad

La herencia permite que algunas propiedades se transmitan a elementos hijos, mientras la especificidad determina qué regla tiene prioridad en caso de conflicto.

Display: Flexbox

Es un diseño unidimensional que organiza elementos en filas o columnas con control preciso de alineación y espacio entre ellos.

Positions

Determinan cómo se posicionan los elementos: static, relative, absolute, fixed y sticky, según su relación con otros elementos o el viewport.

Display: Grid

Es un sistema de diseño bidimensional que organiza contenido en filas y columnas para crear layouts complejos.

Estilos Responsivos

Adaptan el diseño a diferentes tamaños de pantalla utilizando media queries, tamaños relativos y técnicas flexibles. Una de las principales técnicas para lograr esto son la Media Queries
